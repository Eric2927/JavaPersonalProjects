Computer Science

Course Description

As a junior, I am enrolled in the Advanced Computer Science class. This course is taught by Mrs. Angela Taricco. This course focuses on java and implements a total of three major projects over the course of the four terms. The first project is actually the production and maintenance of this website. The second project is the development of a software product and the final project is the development of a mobile application that solves some problem in the real world. The course offers a comprehensive overview of the basics of object-oriented programming and uses the eclipse IDE environment to code.
